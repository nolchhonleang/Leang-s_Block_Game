# Leang's Blocks — Classic Tetris in Python

A clean, fun Tetris clone written in pure Python using only **Tkinter** (no extra packages needed).

Features classic gameplay: rotating pieces, line clearing, scoring, levels that speed up, next-piece preview, pause/resume, and a persistent high score saved to `high_score.json`.


## Features

- 7 classic Tetrominoes with bright colors
- Arrow Keys: Left/Right/Down + Up to rotate
- Increasing speed every 10 lines
- Score, Level, Lines & High Score display
- Pause / Resume button
- Restart anytime
- High score saved between sessions
- Single-file, zero dependencies

## How to Run

```bash
git clone https://github.com/yourusername/leangs-blocks.git
cd leangs-blocks
python leangs_blocks.py
```

Works on Windows, macOS, and Linux with Python 3.6+.

## Controls

| Key       | Action            |
|-----------|-------------------|
| Left      | Move left         |
| Right     | Move right        |
| Down      | Soft drop         |
| Up        | Rotate            |
| Start     | Start / Pause       |
| Restart   | New game          |

## Files

- `leangs_blocks.py` – the complete game (one file)
- `high_score.json` – created automatically to store your best score
- `screenshot.png` – optional preview image
